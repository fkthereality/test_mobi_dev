Datsenko Задание 2.2

Создать массив arr1, заполнить его при помощи цикла числами от 0 до 10. Создать второй массив arr2, заполнить его при помощи цикла квадратами чисел из первого массива arr1.

Задача имеет несколько путей решения, каждый из которых приведет к желаемому результату

При приведелнии к квадрату можно просто умножить i саму на себя arr2[i] = arr1[i] *arr1[i];

но я использовал масштабируемый вариант, для которого легко поменять степень на любую другую при необходимости:

arr2[i] = Math.pow(+i, 2);

Но это не единственные возможные изменения решения.

К примеру, цикл можно организовать через

for (var i = 0; i < 11; i++) { arr1[i] = i; }

, а можно через while или do..while

Значения тоже можно забрать arr2=arr1.slice();

или даже склонировать arr1 в arr2, после чего менять уже копию. В общем, вариантов - масса, предоставлен только один из них.
